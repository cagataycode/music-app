
# spotify clone music app

fullstack music app

# key features

* play/stop songs
* repeat, shuffle
* play next song, previous song
* navigate through the song using slider

# getting started

  ```sh
  npm install npm@latest -g
  ```
  ### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/cagataycode/music-app
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. start the server
   ```sh
   npm run dev
   ```
